# 🌿 BVB Website - Nieuw Ontwerp 2026

Dit is de werkomgeving voor de nieuwe, onafhankelijke website van de Bonsai Vereniging Brabant. We zijn overgestapt van WordPress naar een moderne, statische website voor betere prestaties en volledige creatieve vrijheid.

## 📖 Belangrijke Informatie
- **[PROJECT_HISTORIE.md](file:///d:/BVB/01_Basis/04_Website/04_Nieuw_Ontwerp/PROJECT_HISTORIE.md)**: Hier vind je het volledige logboek van alle gemaakte keuzes, voltooide pagina's en de huidige status. **Lees dit eerst!**
- **[index.html](file:///d:/BVB/01_Basis/04_Website/04_Nieuw_Ontwerp/index.html)**: De hoofdpagina.

## 📁 Bestanden:
- `index.html` t/m `nbs.html`: De verschillende webpagina's.
- `index.css`: De centrale styling (branding, kleuren, lettertypes).
- `*.jpg` / `*.png`: De geoptimaliseerde afbeeldingen voor de site.

## 🚀 Publicatie:
Deze bestanden zijn klaar om geüpload te worden naar **GitHub Pages** (of een andere hosting provider). Er is geen database of WordPress-installatie nodig.
